## 快捷键

| 键 | 功能 |  
| -- | -- |  
| Shift + 1 | 查看整体 |
| Shift + 2 | 查看当前选中 |
| Command + Option + K | 创建组件 |
| Option + 1 | 查看 File |
| Option + 2 | 查看组件 |


## 快速布局

1. F 创建 Frame
2. 将 Frame 变成 Group
3. 给 Group 添加 Auto Layout
4. 设置 Auto Layout 为 Hug Content
