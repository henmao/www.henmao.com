# Vim 快捷键速查

> 前缀键（Leader）已改为 `;`

---

## 1. 模式切换

| 快捷键 | 功能描述 |
|---|---|
| `i` | 在光标前进入插入模式 |
| `a` | 在光标后进入插入模式 |
| `o` | 在下方新建一行并进入插入模式 |
| `O` | 在上方新建一行并进入插入模式 |
| `v` | 进入可视模式（字符选择） |
| `V` | 进入可视行模式（整行选择） |
| `Ctrl + v` | 进入可视块模式（列选择） |
| `Esc` | 返回普通模式 |

---

## 2. 光标移动

| 快捷键 | 功能描述 |
|---|---|
| `h / j / k / l` | 左 / 下 / 上 / 右 |
| `w / b` | 跳到下 / 上一个单词开头 |
| `e` | 跳到当前单词结尾 |
| `0 / $` | 跳到行首 / 行尾 |
| `gg / G` | 跳到文件开头 / 结尾 |
| `{行号} + G` | 跳到指定行，如 `10G` |
| `Ctrl + d / u` | 向下 / 上翻半页 |
| `% ` | 跳到匹配的括号 |

---

## 3. 编辑操作

| 快捷键 | 功能描述 |
|---|---|
| `x` | 删除光标处字符 |
| `dd` | 删除当前行 |
| `{n}dd` | 删除 n 行，如 `3dd` |
| `yy` | 复制当前行 |
| `{n}yy` | 复制 n 行 |
| `p / P` | 在光标后 / 前粘贴 |
| `u` | 撤销 |
| `Ctrl + r` | 重做 |
| `ciw` | 修改光标所在单词（Change Inner Word） |
| `di"` | 删除引号内内容 |
| `>` / `<` | 可视模式下缩进 / 反缩进 |
| `.` | 重复上一次操作 |

---

## 4. 搜索替换

| 快捷键 | 功能描述 |
|---|---|
| `/{keyword}` | 向下搜索关键词 |
| `?{keyword}` | 向上搜索关键词 |
| `n / N` | 跳到下 / 上一个匹配项 |
| `:%s/old/new/g` | 全文替换 |
| `:%s/old/new/gc` | 全文替换（逐个确认） |

---

## 5. 文件与保存

| 快捷键 | 功能描述 |
|---|---|
| `:w` | 保存 |
| `:q` | 退出 |
| `:wq` | 保存并退出 |
| `:q!` | 强制退出不保存 |

---

## 6. 分屏操作

| 快捷键 | 功能描述 |
|---|---|
| `:vsp` | 左右垂直分屏 |
| `:sp` | 上下水平分屏 |
| `Ctrl + w + h/j/k/l` | 在分屏间移动光标 |
| `Ctrl + w + q` | 关闭当前分屏 |

---

## 7. 插件搜索（Telescope）

| 快捷键 | 功能描述 |
|---|---|
| `;p` | 搜索项目文件 |
| `;/` | 搜索文件内容（Live Grep） |
| `;b` | 搜索已打开的 Buffer |
